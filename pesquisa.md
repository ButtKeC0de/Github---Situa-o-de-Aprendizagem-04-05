Pesquisa XAMPP com PHP


O que é XAMPP?
O XAMPP é um pacote de software livre que facilita a criação e manipulação de servidores web locais.Isso significa que os devs podem desenvolver e testar sistemas de maneira segura em um ambiente controlado antes que possa ser publicado.
Isso é estritamente importante pois desse modo os programadores podem ter ousadia maior em seus projetos sem muitos riscos,pois este é o local propenso á erros,pois podem ser corrigidos sem julgamento ou interferência externa. Podemos afirmar que é como uma simulação do projeto,que portanto permite testar inúmeros protótipos e passar a limpo para aí sim ser lançado.
Quais são seus principais componentes?
Apache(Servidor): 
O Apache é o servidor HTTPS do XAMPP. É ele responsável por permitir ao XAMPP a criar páginas Web nos navegadores. Ele é essencial pois ele: Interpreta e executa os arquivos HTML,PHP,CSS e entre muitos outros, respondendo às requisições do usuário.
MySQL/MariaDB: 
O XAMPPS conta com o MySQL ou o MariaDB como sistema de gerenciamento de banco de dados.Isso possibilita o armazenamento e organização de dados para a aplicação WEB.
PHP:
PHP é uma linguagem de programação focada em desenvolvimento WEB.No XAMPPS ela é utilizada para criar e configurar scripts que interagem com o Banco de Dados de forma dinâmica.
Perl:
O Perl é uma linguagem de script poderosa e versátil e, dentro do XAMPP, usado para automação de tarefas, manipulação de textos e desenvolvimento de aplicações web.
