Termos de Uso

Sistema de Monitoramento Ferroviário

1. Aceitação dos Termos

Ao acessar e utilizar este sistema de monitoramento de mobilidade urbana inteligente, o usuário declara estar ciente e de acordo com as diretrizes de segurança, integridade e uso operacional definidas pela empresa proprietária do sistema.

2. Acesso e Segurança

O acesso ao sistema é restrito a administradores devidamente autorizados mediante autenticação via e-mail e senha.

É responsabilidade do usuário manter a confidencialidade de suas credenciais de acesso, sendo vedado o compartilhamento de senhas.

O sistema registra a sessão do usuário desde o login até o momento do logout.

3. Uso dos Dados e Monitoramento

Os dados capturados pelos sensores IoT (velocidade, localização, consumo de energia e falhas) são de propriedade da empresa e destinados exclusivamente para fins de gestão operacional, manutenção preditiva e segurança.

É proibida a manipulação indevida dos dados ou a alteração das configurações de sensores sem a devida autorização no módulo administrativo.

4. Responsabilidades

O administrador é responsável por toda e qualquer alteração realizada no sistema, incluindo o cadastro e a exclusão de sensores, respeitando as restrições de integridade de dados.

O sistema proíbe a exclusão de sensores que possuam registros de dados associados para garantir a precisão histórica das informações.

5. Alterações e Finalização

A empresa reserva-se o direito de modificar funcionalidades do sistema conforme a necessidade de melhoria na mobilidade urbana.

Ao sair do sistema (logout), a sessão é automaticamente encerrada, sendo necessário nova autenticação para acesso.
