# Situação de Aprendizagem Ferrorama

## Sobre o Projeto

Este projeto foi desenvolvido como uma Situação de Aprendizagem com foco em mobilidade urbana inteligente.  
A proposta consiste na criação de um protótipo funcional de um sistema ferroviário monitorado em tempo real, utilizando sensores IoT instalados nos trilhos e nas locomotivas para coletar informações importantes do sistema ferroviário.

Os dados a serem capturados incluem:
- Velocidade dos trens
- Localização em tempo real
- Consumo de energia
- Detecção de falhas

Todas essas informações são enviadas para um banco de dados central, permitindo o desenvolvimento de dashboards e relatórios operacionais voltados para:
- Desempenho do sistema
- Manutenção preditiva
- Segurança operacional
- Apoio à tomada de decisões

O projeto busca seguir boas práticas de programação, integração de sistemas e gestão de dados.

## Tecnologias Utilizadas até o momento:

- HTML
- CSS
- JavaScript
- Bootstrap 

## Funcionalidades

- Monitoramento ferroviário em tempo real
- Captura de dados por sensores IoT
- Armazenamento de informações em banco de dados
- Geração de dashboards operacionais
- Relatórios de desempenho e manutenção
- Detecção de falhas no sistema
